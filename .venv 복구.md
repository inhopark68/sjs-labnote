A) 깨진 가상환경 삭제
PowerShell:bat

rmdir /s /q .venv

B) 가상환경 다시 생성 (이번엔 중단하지 마세요)
PowerShell:bat

py -3.11 -m venv .venv

python -m venv .venv

C) 가상환경 활성화 (사용 중인 터미널에 따라 다름)

✔ CMD
.venv\Scripts\activate

✔ PowerShell
.venv\Scripts\Activate.ps1

✔ Git Bash
source .venv/Scripts/activate

D) venv 안에서 pip 업그레이드 + 패키지 설치
python -m pip install --upgrade pip
